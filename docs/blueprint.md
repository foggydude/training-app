# **App Name**: Adaptive Training Planner

## Core Features:

- Google Login: Secure user authentication using Google sign-in.
- Intervals.icu Integration: Connects to Intervals.icu via API to retrieve and synchronize training data.
- Race Management: Create, edit, and manage races with specific details such as sport, date, distance, and goals.
- Training Plan Generation: Generates initial and updated training plans based on user history, race goals, and compliance using custom algorithms.
- Weekly Plan Adaptation: Dynamically adjust the upcoming week's training plan, increasing or decreasing volume based on recent performance, as informed by intervals.icu data.
- Forecast Computation: Computes a predicted race time and confidence interval based on the consistency of the training log.
- Calendar View: Visualize the training plan in a calendar format, showing workouts as blocks colored by sport and type.

## Style Guidelines:

- Primary color: Deep sky blue (#3399FF) to reflect energy, drive, and athletic excellence.
- Background color: Light gray (#F0F0F0) to provide a clean and neutral base.
- Accent color: Purple (#9933FF) for highlights, progress indicators, and CTAs to signify the data-driven analysis.
- Body and headline font: 'Inter', a sans-serif font, is suitable for both headlines and body text due to its modern and neutral design.
- Use simple, outlined icons for workout types, sports, and UI actions to maintain a clean interface.
- Utilize a calendar-focused layout, with workouts displayed as blocks and expandable details in a right-side panel.
- Subtle transitions when the user loads, changes or syncs the weekly data.
